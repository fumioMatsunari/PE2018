## 今回の特徴
・　タブレット・プログラミング
小学生などを見ていると、新しいノートパソコンを買ってもらえる子もいますし、IT関係者の子供さんは親のお古でといった感じもありますが、わざわざパソコンを買うというのには抵抗が多い。タブレットの方がハードルが低い。
子どものプログラミングもタブレットで出来る方向にある気がします。もう少し、タブレットでもいいよといえば、プログラミングのハードルが低くなると思っています。
・　CoderDojo天白の4周年記念イベントになります。
回数としてはCoderDojo天白だけでも200回になりますね。そのほかにScratchDojo、天白生涯学習センターでの会もやっています。思えば遠くへきたもんだ(^^♪
・　初めて、名工大の白松先生の研究室と共催にしました。ゆめ基金に聞けば、助成金が共催相手に出ていかなければいいというような感じでした。いま、Code for Aichiとコンタクトしていて、そんなこともこれから考えなければいけないことかななどと思います。
##### 1. タブレット端末でプログラミング！
　最近、ノートパソコンは持たないけれど、タブレットを使っている家庭が多いのに気が付きます。プログラミングもだんだんとタブレットでもできるようになってきています。今回は、4種類のプログラミングを扱いますが、ノートパソコンとタブレット端末(iPad)の両方を使います。
##### 2. 今回はCoderDojo天白の4周年記念行事となります！
　いろいろ、あったけど、Scratchをはじめて5年、Dojoはじめて4年、ただの自己満足ですが・・・
##### 3. 白松研究室との共催
　また、名工大の人とのつながりができれば有難い
##### 4. 全体を通してのテーマを作ってみました
 
## 構成
##### 午前、午後の2部構成になります。
1. 午前はじゃんけんの作品を作ります
2. 午後はメッセージを使います。
・　午前はじゃんけんを組み入れる
　　午前の最後はじゃんけん大会
・　午後はメッセージをいろんな形で使う
　　ピョンキーを入れたのはMESHを使いたいから（古い？）
　　最後に集団発表
　　スクラッチは難しいかな
・　出来たらタブレット（iPad)を使いたい
##### アプリ・機器は次の4種類です
1. Scratch3.0で体験　・・・ノートパソコン、iPad を使用
2. ピョンキー　・・・　iPadを使用
3. micro:bit Scratch  ・・・ノートパソコン、iPad
4. Viscuit　　・・・　ノートパソコン、iPad
