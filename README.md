# プログラミング体験で可能性を探そう！
## 共催　名古屋工業大学　白松研究室
## 後援名義　名古屋市教育委員会
### 開催場所　名古屋工業大学 21号館１F2111室
### 開催日時　2018年10月27日（土）9:00～16:30

プログラミングの入門的なワークショップです。

最近、ノートパソコンを持たないけど、iPadなどのタブレットを持っている人が多いことに気が付きました。

タブレットを用いたプログラミングを主体に行う予定です。

また、facebookでは皆さんの進捗がはっきりしないので、GitHubを用いて進行管理をしたいと思っています。

## 今回の特徴
##### 1. タブレット端末でプログラミング！
　4つのアプリを使いますが、ノートパソコンとタブレット端末(iPad)の両方を使います。
##### 2. 今回はCoderDojo天白の4周年記念行事となります！
　いろいろ、あったけど、Scratchをはじめて5年、Dojoはじめて4年、ただの自己満足ですが・・・
##### 3. 白松研究室との共催
　また、名工大の人とのつながりができれば有難い
##### 4. 全体を通してのテーマを作ってみました
 
## 構成
##### 午前、午後の2部構成になります。
1. 午前はじゃんけんの作品を作ります
2. 午後はメッセージを使います。
##### アプリ・機器は次の4種類です
1. Scratch3.0で体験　・・・ノートパソコン、iPad を使用
2. ピョンキー　・・・　iPadを使用
3. micro:bit Scratch  ・・・ノートパソコン、iPad
4. Viscuit　　・・・　ノートパソコン、iPad
